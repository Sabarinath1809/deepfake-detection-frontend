Project Overview

This final‑year project presents a smart, multi‑stage system for detecting deepfake images and videos. The pipeline combines face alignment, feature extraction, temporal analysis, and visual explanation to deliver a clear verdict on media authenticity.

How It Works

  1. MTCNN locates & aligns faces in each frame.
  2. XceptionNet extracts deep visual features.
  3. LSTM captures temporal inconsistencies across frames.
  4. Grad‑CAM highlights manipulated regions for transparency.

Technology Stack

Frontend

  • HTML, CSS (custom dark theme)
  • Vanilla JavaScript (file upload, progress, API calls)
  • GitHub Pages for hosting

Backend

  • Python 3 – Flask API
  • MTCNN, Keras (XceptionNet), TensorFlow (LSTM)
  • OpenCV, Grad‑CAM utils

Datasets

  • FaceForensics++
  • DFDC (Deepfake Detection Challenge)
  • Celeb‑DF v2
