About This Project

Deepfake Detect is an AI-driven platform built to identify and flag manipulated media — whether image or video. With the rise of synthetic content, it's essential to have tools that can help verify digital media authenticity.

This system utilizes powerful deep learning models including MTCNN, XceptionNet, LSTM, and advanced techniques like Grad-CAM and Optical Flow to detect deepfakes with high accuracy.

It is built as part of a final year academic project with the goal of enhancing public awareness and providing practical AI-based solutions to combat misinformation.
